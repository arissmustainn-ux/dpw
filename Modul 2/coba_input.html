UJI COBA INPUT HTML

Uji coba 1

Pilih Jenis Kelamin :



Kesimpulan Uji coba 1:

Atribut "name" dengan nilai yang sama untuk semua radio button memungkinkan pengguna hanya memilih satu opsi. Jika atribut "name" berbeda, maka semua radio button dapat dipilih secara bersamaan (tidak eksklusif).

Uji coba 2

Pilih Hobi :



Kesimpulan Uji coba 2:

Checkbox memungkinkan pengguna memilih lebih dari satu opsi. Atribut "name" yang sama akan mengelompokkan checkbox dalam satu kategori, namun tetap bisa memilih multiple.

Uji coba 3




Kesimpulan: Tombol "Click Me!" akan memunculkan alert dengan pesan "Hello World!" ketika ditekan.

Uji coba 4




Kesimpulan: Type datetime-local menampilkan pilihan tanggal dan waktu, type month menampilkan pilihan bulan dan tahun, type email akan memvalidasi format email yang benar.

Uji coba 5




Kesimpulan: Type image berfungsi sebagai tombol submit bergambar, type file untuk mengupload file, type hidden menyimpan data yang tidak terlihat oleh user.

Uji coba 6



Kesimpulan: Input number memiliki batasan min dan max, jika min diubah menjadi -50 maka input bisa menerima angka negatif. Input range menggunakan slider untuk memilih nilai dalam range tertentu.

Uji coba 7




Kesimpulan: Type url memvalidasi format URL, type search untuk pencarian, type tel dengan pattern memvalidasi format nomor telepon sesuai pattern yang ditentukan.

Uji coba 8



Kesimpulan: Type time untuk memilih jam, type week untuk memilih minggu dalam setahun.
