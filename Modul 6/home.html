Mengapa Bootstrap?

⚡
Cepat

Bootstrap menyediakan komponen siap pakai sehingga pengembangan lebih cepat.

📱
Responsif

Tampilan otomatis menyesuaikan ukuran layar dari mobile hingga desktop.

🎨
Indah

Desain modern dan konsisten tersedia tanpa perlu menulis banyak CSS dari nol.

Cara Menambahkan Bootstrap

1. Via CDN (paling mudah)

Tambahkan kode berikut di dalam tag <head> HTML:

<link rel="stylesheet"
 href="https://cdn.jsdelivr.net/
 npm/bootstrap@5.3.2/dist/css/
 bootstrap.min.css">

Tambahkan script JS sebelum </body>:

<script src="https://cdn.jsdelivr.net/
 npm/bootstrap@5.3.2/dist/js/
 bootstrap.bundle.min.js"></script>
2. Via Download
  1. Kunjungi https://getbootstrap.com
  2. Klik tombol Download
  3. Ekstrak file zip yang diunduh
  4. Salin folder css dan js ke proyek kamu
  5. Link file lokal di HTML kamu
